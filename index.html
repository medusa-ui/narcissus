Pages are simply written in HTML and enhanced with expressions to allow for efficient DOM rewrites. Interactivity is handled serverside through WebSockets. The event-driven serverside code is written around non-blocking Spring Reactive.

That means, for example, you get the following for free:

- Clicking a button triggers an immediate event on the server

- A database change is reflected on the page without AJAX

<!DOCTYPE html> <html lang="en"> <body> <h1>Hello [$name]</h1> <p>Counter: [$my-counter]</p> <button m-click="increase(1)">Increase value</button> </body> </html>
